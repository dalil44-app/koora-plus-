# 🤝 دليل المساهمة

شكراً لاهتمامك بالمساهمة في مشروع كورة بلس!

## كيفية المساهمة

### 1. الإبلاغ عن الأخطاء (Bug Reports)
- استخدم قسم Issues
- اشرح المشكلة بوضوح
- أضف خطوات إعادة إنتاج الخطأ
- أرفق لقطات شاشة إن أمكن

### 2. اقتراح ميزات جديدة (Feature Requests)
- استخدم قسم Issues مع وسم `enhancement`
- اشرح الفكرة والفائدة المتوقعة
- قد تكون هناك نقاشات لتحسين الفكرة

### 3. طلبات الدمج (Pull Requests)
1. انسخ المستودع (Fork)
2. أنشئ فرعاً جديداً (`git checkout -b feature/amazing-feature`)
3. ادفع التغييرات (`git commit -m 'Add amazing feature'`)
4. ادفع للفرع (`git push origin feature/amazing-feature`)
5. افتح Pull Request

## معايير الكود

- استخدم تعليقات بالعربية أو الإنجليزية
- حافظ على تنظيم الكود
- اختبر التغييرات قبل الإرسال
- احترم تصميم الواجهة الحالي

## أسئلة؟

يمكنك التواصل عبر Issues أو البريد الإلكتروني.

شكراً! ⚽
