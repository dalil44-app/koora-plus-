# ⚽ كورة بلس | Koora Plus

موقع عربي متكامل لعشاق كرة القدم - يعرض المباريات، المراجعات، القوائم، والأخبار الرياضية.

## 🎯 المميزات

- ⚡ **صفحة تحميل أنيقة** - لوغو متحرك يختفي تلقائياً
- 🏠 **واجهة رئيسية** - تصميم عصري داكن اللون
- ⚽ **تبويب المباريات** - شبكة بوسترات 2×2 مع إمكانية التمرير
- ⭐ **تبويب المراجعات** - مراجعات المستخدمين مع الصور والتقييمات
- 📌 **تبويب القوائم** - قوائم مباريات مختارة من المستخدمين
- 📰 **تبويب الأخبار** - آخر الأخبار الرياضية
- 🔍 **بحث** - نموذج بحث منبثق
- 👤 **تسجيل الدخول** - نموذج مصادقة بسيط
- 📱 **تصميم متجاوب** - يعمل على جميع الأجهزة

## 🚀 طريقة التشغيل

### الطريقة الأولى: مباشرة
افتح ملف `index.html` في أي متصفح حديث.

### الطريقة الثانية: سيرفر محلي
```bash
# باستخدام Python
python -m http.server 8000

# باستخدام Node.js
npx serve .

# باستخدام PHP
php -S localhost:8000
